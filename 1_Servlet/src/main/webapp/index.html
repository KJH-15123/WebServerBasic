서블릿 이란?
웹 서비스를 위한 "자바 클래스"를 말하며 자바를 사용하여 웹을 만들기 위해 필요한 기술
- 사용자의 요청을 받아 처리하고 그에 해당하는 응답페이지를 만들어 다시 사용자에게 전송하는 역할을 하는 자바클래스
Controller의 기능을 수행한다.
- 웹에서 동적인 페이지를 JAVA로 구현할 수 있도록 도와주는 서버측 프로그래(WAS 서버에서 구동)
JAVA클래스에서 웬페이지 구현을 위한 HTML이 들어간 구조라고 할 수 있음(java코드로 HTML구현)
